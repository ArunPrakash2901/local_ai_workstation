# Execution Lane Examples

Execution Lane examples are documentation-only for now.

Future example execution inputs should be copied or referenced from:

```text
discovery_lane/execution_queues/positive_path_example_execution_queue.json
```

Do not duplicate large generated artifacts here unless a future test or fixture needs a stable snapshot.

The positive-path Discovery Lane fixture proves that an example queue can reach:

```text
READY_FOR_EXECUTION_LANE
```

Execution Lane still does not execute worker prompts, create branches, checkout branches, commit, push, or merge.
