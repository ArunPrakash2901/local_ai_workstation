# PRD Brief Contract

A PRD brief is a deterministic product planning artifact generated from approved Discovery Lane handoffs.

## Required Sections

- Product Name
- Product Thesis
- Target Users
- Problem Statement
- Goals
- Non-Goals
- User Journeys
- Core Requirements
- Release Scope
- Acceptance Criteria
- Risks
- Open Decisions

## Rules

- Do not expand scope beyond approved Discovery handoffs.
- Use `NEEDS_HUMAN_DECISION` for missing product, user, or release decisions.
- Do not claim implementation or validation has happened.
