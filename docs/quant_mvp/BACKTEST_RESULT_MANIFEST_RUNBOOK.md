# Backtest Result Manifest Runbook

## Purpose
The Result Manifest captures the final metadata output of a backtest run. For Milestone Q11, this manifest strictly captures `synthetic_smoke_test_only` runs.

## Manifest Fields
- `synthetic_fixture`: Must be `True`.
- `strategy_logic_used`: Must be `False`.
- `metrics`: Contains the deterministic output generated by the toy skeleton engine.

## Forbidden Interpretations
Under no circumstances may the results within this manifest be interpreted as:
- A profitable strategy.
- A trading signal.
- Grounds for live or paper execution.
All artifacts generated must explicitly state that no real market data was used.

## Execution
The manifest is automatically drafted and validated when executing the `synthetic-smoke` command from the `backtest_cli.py`.

```powershell
python scripts/quant/backtest_cli.py synthetic-smoke `
  --fixture scratch/quant_backtests/synthetic_price_fixture.json `
  --write
```
